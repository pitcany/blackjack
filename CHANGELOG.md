## [0.1.0] - 2025-01-01

### Added
- Initial release of Blackjack Card Counter
- Modular package architecture
- Hi-Lo card counting system
- Basic strategy recommendations
- Betting strategy based on true count
- Full blackjack gameplay (hit, stand, double, split)
- Customizable deck count (1-8 decks)
- Bankroll management
- Interactive Pygame UI
- Poetry packaging with PEP 621 compliance
- MIT License
- Development tools (black, flake8, mypy, pytest, isort)

### Package Structure
- `constants.py` - Configuration and constants
- `card.py` - Card and deck logic
- `ui.py` - UI components (Button, TextInput)
- `strategy.py` - Strategy calculations
- `game.py` - Main game orchestrator
