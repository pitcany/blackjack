# Examples

This directory contains example scripts and alternative ways to use the blackjack package.

## Standalone Game

**File:** `standalone_game.py`

The original standalone version of the game. This is a single-file implementation that works without installing the package.

**Usage:**
```bash
python examples/standalone_game.py
```

This is useful if you want to:
- See the complete game in one file
- Understand how everything works together
- Use without installing the package
- Learn Pygame basics

## Using the Package

For normal usage, install and use the package:

```bash
# Install
poetry install

# Run
poetry run blackjack
```

See the main [README.md](../README.md) for full documentation.
